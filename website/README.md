# ReY Private Website (Viewer Platform)

This directory contains the viewer-facing web interface styled as a modern React dashboard.

## Folder Organization

- **[`/src/components`](file:///C:/Users/yasao/.gemini/antigravity/scratch/ReY-Platform/website/src/components)**: Reusable components and overlays.
- **[`/src/layouts`](file:///C:/Users/yasao/.gemini/antigravity/scratch/ReY-Platform/website/src/layouts)**: Responsive wrapper routing structures (desktop sidebars, mobile bottom navigation bar).
- **[`/src/pages`](file:///C:/Users/yasao/.gemini/antigravity/scratch/ReY-Platform/website/src/pages)**: Core page views including login, map display, rich status view, and AI dialog box.
- **[`/src/index.css`](file:///C:/Users/yasao/.gemini/antigravity/scratch/ReY-Platform/website/src/index.css)**: Unified CSS color tokens, glassmorphism panel styles, fonts, and animation wrappers.

## Local Development

Start the development server using npm:
```bash
npm run dev
```

Refer to the global **[`/documentation/DEPLOYMENT.md`](file:///C:/Users/yasao/.gemini/antigravity/scratch/ReY-Platform/documentation/DEPLOYMENT.md)** for instructions on publishing as a Cloudflare Pages asset.
